## Description
This project is an AI-assisted study tool designed to help students organize notes,
generate practice questions, and track learning progress. The goal is to make studying
more efficient by combining note management with AI-powered insights.

## Features
- Upload and organize study notes
- AI-generated practice questions
- Progress tracking
- Searchable knowledge base

  ## Technologies Used
- Python
- JavaScript
- HTML/CSS
- SQL
- OpenAI API

## Usage
After launching the application, users can upload study notes,
generate quizzes, and track their progress through the dashboard.
